# 🏙️ City-Pulse AI 2.0 Ultra
### *An AI-Driven Predictive Safety Analytics Platform*

## 🚀 Overview
City-Pulse AI 2.0 Ultra is a next-generation safety dashboard designed to transform raw crime data into actionable intelligence. By leveraging historical datasets from **Data.gov.in**, the platform provides visual heatmaps, regional comparisons, and AI-powered safety predictions up to the year 2027.

## ✨ Key Features
- **Predictive Analytics:** Uses linear variance models to forecast safety trends.
- **National Safety Heatmap:** Interactive geospatial visualization of safety zones.
- **Battle Mode:** Side-by-side comparative analysis of different states/cities.
- **JUHI AI Assistant:** A voice-enabled LLM (Llama-3.3 via Groq) that interprets complex data for users.
- **Future Ready:** Designed for real-time API integration and SOS functionalities.

## 🛠️ Tech Stack
- **Language:** Python
- **Frontend:** Streamlit
- **AI Model:** Llama-3.3-70b (Groq LPU Inference)
- **Data Science:** Pandas, NumPy, Plotly
- **Speech:** gTTS (Google Text-to-Speech)

## 📁 Project Structure
- `app.py`: Main application code.
- `requirements.txt`: List of dependencies.
- `data/`: Contains the safety datasets.
- `assets/`: Project screenshots and images.
